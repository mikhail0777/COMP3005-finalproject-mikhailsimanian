## üìå Overview

This project implements a fully functional **Health & Fitness Club Management System** using:

- **PostgreSQL** (Relational DBMS)
- **Python (CLI Application)** using `psycopg2-binary`
- **SQL schema** including:
  - Tables
  - Primary & Foreign Keys
  - CHECK constraints
  - Index
  - View
  - Trigger
- **Minimum 8 required operations** across:
  - Member
  - Trainer
  - Admin

The system simulates real fitness club operations such as:

- Member registration & health tracking  
- Personal training session scheduling  
- Group class registration with capacity limits  
- Trainer schedule viewing & member lookup  
- Admin class scheduling, room management, and invoicing  

Everything is persistent in PostgreSQL.
